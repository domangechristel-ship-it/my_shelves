# 📚 My Shelves

**My Shelves** est un projet de recommandation de livres nouvelle génération, qui va au-delà des filtres classiques (auteur, genre…) en exploitant le contenu des livres et des avis utilisateurs.

## 🚀 Objectif

Proposer une expérience de découverte de livres plus intelligente et immersive, basée sur :
- les émotions ressenties
- le rythme du livre
- le contexte géographique
- le contenu des reviews utilisateurs

## 🧠 Concept

Contrairement aux systèmes classiques, **My Shelves** analyse :
- 📖 les descriptions de livres
- 💬 les reviews utilisateurs
- 🌍 les lieux où se déroulent les histoires

pour offrir des recommandations plus riches et personnalisées.

## ✨ Fonctionnalités (en cours / prévues)

- 🗺️ **Carte interactive**
  Explorer les livres par pays ou région

- 😊 **Recherche par émotions & critères**
  Trouver un livre selon l’humeur, le rythme, le nombre de pages…

- 🔍 **Recommandation de livres similaires**
  Basée sur contenu + embeddings

- 🤖 **Chatbot intelligent (RAG)**
  Exemple : *“Je cherche un livre pendant la seconde guerre mondiale en Italie”*
